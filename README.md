# nath-scripts

**Is Scripts Se Aap Ek Command Se Ubuntu+xfce Desktop Start Kar Sakte Hai**

### Niche Ke Commands Copy Kar Ke Termux Me Paste Kare

1. termux ke packages update karne ke liye..

```bash
pkg update && pkg upgrade -y
```

2. wget install karne ke liye (important)..

```bash
pkg install wget -y
```

3. ye command start script Download karne ke liye hai..

```bash
wget https://raw.githubusercontent.com/sahu-dev-hub/nath-scripts/refs/heads/main/setup.sh
```
4. ye script ko setup karne ke liye hai..

```bash
bash setup.sh
```

<b>NOTE:</b> Jab Aap Ko Kahe "Apna Shortcu Name Type Kare" Tab Aap Wahi Name Ya Word Type Kare Jis Name/Word Se App Desktop Start Karna Chahte Hai..

   
5. Ab Ek Baar Termux Ko Restart Kare..

   ```bash
   exit
   ```
